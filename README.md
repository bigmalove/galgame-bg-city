# 内置图包·现代都市

galgame 界面插件内置背景图包（30 个场景，2K webp）。

## 场景清单

街道、公寓、咖啡厅、办公室、写字楼、商场、地铁站、公园、餐厅、酒吧、医院、健身房、电影院、游乐园、楼顶、雨夜街头、书店、超市、天桥、港口、机场、停车场、酒店、大学、教堂、KTV、阳台、加油站、十字路口、美术馆

## 在插件中使用

资源管理器 → 导入 → 内置背景图包，或手动「导入远程链接JSON」填入：

```
https://cdn.jsdelivr.net/gh/bigmalove/galgame-bg-city@v1.1.0/remote_assets.json
```
